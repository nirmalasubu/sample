OnDemand Tools
======

ODT is a distribution platform for delivering digital content and data around it's content.

### Documentation

- [http://docs.ondemandtools.apiary.io](http://docs.ondemandtools.apiary.io)

- .NET Core
- Mongo (via mLab)
- RabbitMQ (via CloudAMQP)
- NancyFx
- Node
